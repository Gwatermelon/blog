# Ge Zhang · 技术笔记

这是一个基于 [Hugo](https://gohugo.io/) 和 PaperMod 主题构建的个人技术博客，主要记录 张革平时工作中的对于各类技术栈的学习过程和思考。

## 在线访问

本项目托管在 **Cloudflare Pages**，内容持续更新并发布到以下地址：

**每日产出：<https://blog-shf.pages.dev>**

## 项目结构

```text
content/     博客文章与研究笔记
layouts/     自定义页面模板
assets/      自定义样式与静态资源
themes/      Hugo 主题
hugo.toml    站点配置
```

## 部署说明

代码更新后由 Cloudflare Pages 构建并发布，线上站点以 <https://blog-shf.pages.dev> 的内容为准。
